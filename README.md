# Satang Corp Assignment
### Tech Stacks
- React (Vite)
- Typescript
- Redux + Redux Toolkit
- Redux Saga

### How to run
    With Docker
        - run docker application
        - docker build -t satang-dev .
        - docker run --rm  --name satang-dev -p 5173:5173 satang-dev:latest
    With NPM
        - npm install --force
        - npm run start
